# New Client Install Checklist

## Setup

- [ ] Copy `03-client-installs/client-name-template`
- [ ] Rename the copy to the client slug
- [ ] Fill `CLIENT_PROFILE.md`
- [ ] Add approved script to `CLIENT_SCRIPT.md`
- [ ] Add FAQs and knowledge base entries to `CLIENT_FAQ.md`
- [ ] Add booking rules to `CLIENT_BOOKING_RULES.md`
- [ ] Add escalation and forbidden topics to `CLIENT_ESCALATION_RULES.md`
- [ ] Add notification recipient to `CLIENT_NOTIFICATION_SETTINGS.md`

## Review

- [ ] Confirm the business name and greeting
- [ ] Confirm required lead fields
- [ ] Confirm forbidden topics
- [ ] Confirm callback and escalation wording
- [ ] Confirm approved static audio assets if used

## Test

- [ ] Run local test
- [ ] Deploy to Vercel when approved
- [ ] Place live test call
- [ ] Confirm Resend summary
- [ ] Mark client ready
