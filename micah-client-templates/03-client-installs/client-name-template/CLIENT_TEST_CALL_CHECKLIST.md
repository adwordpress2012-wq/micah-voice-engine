# Client Test Call Checklist

## Before Calling

- [ ] Client profile complete
- [ ] Script complete
- [ ] FAQ complete
- [ ] Booking rules confirmed
- [ ] Escalation rules confirmed
- [ ] Notification email confirmed
- [ ] Static MP3 assets verified as approved Micah voice if used

## Test Call

- [ ] Greeting uses the correct client business name
- [ ] Micah sounds like the approved female Aussie voice
- [ ] Common FAQ answer is correct
- [ ] Unknown question is escalated
- [ ] Lead fields are captured once
- [ ] Confirmations use "is that right?"
- [ ] No forbidden topic is answered directly
- [ ] Call closes warmly

## After Call

- [ ] Resend summary received
- [ ] Summary is accurate
- [ ] Missing details are marked clearly
- [ ] Logs show expected voice path
- [ ] Client is approved for go-live
